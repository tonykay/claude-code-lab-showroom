Reference / Commands / CLI Flags

CLI Flags

Flags passed when launching Claude Code from the terminal.

Core

  • --print — Non-interactive mode
  • --output-format — text, JSON, or streaming JSON

Session

  • --continue — Resume most recent session
  • --resume [id] — Resume with picker
  • --fork-session — Fork an existing session

Model

  • --model — Aliases or full model IDs
  • --effort — low to max
  • --fallback-model — Fallback model

Permissions

Four modes: default, acceptEdits, plan, bypassPermissions

--dangerously-skip-permissions for strongest automation.

Tool Control

  • --allowed-tools — Allow specific tools (glob patterns)
  • --disallowed-tools — Block specific tools
  • --add-dir — Add working directories

Advanced

  • --mcp-config — MCP config file
  • --system-prompt — Custom system prompt
  • --append-system-prompt — Append to system prompt
  • --bare — Minimal mode
