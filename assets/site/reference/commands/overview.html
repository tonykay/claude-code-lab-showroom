Commands Overview

Two command systems: CLI flags (before launch) and slash commands (during session).

CLI flags configure at launch (model, output format, permissions). Slash commands manage runtime (memory, models, commits).

Keyboard Shortcuts

ShortcutAction
Ctrl+CInterrupts (preserves conversation)
Ctrl+DTerminates session
Ctrl+LClears display
Arrow keysNavigate history
TabAutocomplete

Subcommands

  • mcp — MCP server management
  • doctor — Diagnostics
  • update — Update Claude Code

Help

Use /help in session or claude --help in terminal.
