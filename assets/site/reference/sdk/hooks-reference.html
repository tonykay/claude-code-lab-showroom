Hooks Reference

Detailed reference for hooks: commands, endpoints, prompts, and callbacks at lifecycle events.

Configuration

Settings files under the hooks key. Each has event type, optional matcher, and hook definitions.

Hook Types

TypeDescription
CommandShell command (bash/PowerShell)
PromptLLM-based decision
AgentAgentic loop with tool access
HTTPPOST to endpoint

Events

EventDescription
PreToolUseBefore tool execution
PostToolUseAfter tool execution
PermissionRequestPermission requested
PermissionDeniedPermission denied
StopBefore conclusion
StopFailureStop failed
SessionStartSession beginning
SessionEndSession ending
SubagentStartSub-agent launched
SubagentStopSub-agent finished
FileChangedFile modified
CwdChangedWorking directory changed
ConfigChangeConfiguration changed

Exit Codes

  • 0 = success
  • 2 = block/special behavior
  • other = error without blocking

Async Hooks

Return {"async": true} for non-blocking background execution.
