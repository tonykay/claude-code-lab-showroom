SDK Overview

The SDK uses a subprocess model where the host spawns Claude with JSON message exchange.

Output Formats

  • text — Plain text
  • json — JSON response
  • stream-json — Streaming JSON (SDK requirement)

Initialization

Initialize with initialize control request, returns available commands, models, agents, and account info.

Message Types

  • system
  • assistant
  • tool_progress
  • result
  • stream_event

Features

  • Custom subagents
  • Hook callbacks
  • Session management (transcripts, forking, saved sessions)

Use Cases

  • IDE integration
  • CI/CD automation
  • Headless background agents
