Permissions API

Programmatic control over Claude Code's permission system.

Five Modes

  • default
  • acceptEdits
  • bypassPermissions
  • plan
  • dontAsk

Configuration Methods

  • CLI flags
  • Settings files
  • /permissions command
  • SDK control requests

Rules

Pattern matching with:

  • Glob for bash commands
  • Absolute paths for files
  • mcp__servername for MCP tools
  • Agent(Explore) for agent types

Deny rules always supersede allow rules.

Decision Pipeline

  1. Deny rules
  2. Ask rules
  3. Tool checks
  4. Safety checks
  5. Mode evaluation
  6. Allow rules
  7. Default behavior

Protected Directories

.git/, .claude/, .vscode/ always trigger prompts even in bypass mode.

SDK Integration

PermissionUpdate objects modify rules across scopes.
