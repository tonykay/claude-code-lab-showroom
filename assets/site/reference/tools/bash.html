Reference / Tools / Bash

Bash

Executes shell commands in a persistent session.

Working directory persists between commands; shell state reinitializes each call.

Parameters

ParameterRequiredDescription
commandYesShell command to execute
timeoutNoDefault 120000ms, max 600000ms
descriptionNoHuman-readable description
run_in_backgroundNoRun asynchronously

Behavior

  • Compound commands parsed and checked independently
  • Output truncated if too large
  • Prefer dedicated tools (Read, Edit, Glob, Grep) over bash for file operations

Permissions

Prompts in default mode; auto-approved in acceptEdits for allowed rules.
