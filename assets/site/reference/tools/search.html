Search

File pattern matching and content search tools.

Glob

Fast file pattern matching sorted by modification time, capped at 100 files.

  • Supports patterns like **/*.js or src/**/*.ts
  • Always auto-approved
  • Excludes VCS directories

Grep

ripgrep-based content search with three output modes:

ModeOutput
files_with_matchesFile paths only (default)
contentMatching lines with context
countMatch counts per file
  • Multiline with multiline: true
  • Literal braces need escaping
  • Always auto-approved

Both tools exclude .git, .svn, .hg, .bzr, .jj, .sl directories.
