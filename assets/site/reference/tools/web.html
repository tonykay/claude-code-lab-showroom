Web

Tools for fetching web content and searching the internet.

WebFetch

  • Fetch URL and convert HTML to markdown
  • Processed with secondary model
  • 15-minute cache
  • Handles redirects
  • Read-only operation

WebSearch

  • Search web for current information
  • Returns titles, snippets, and URLs
  • US-only
  • Domain filtering available
  • Citations required in responses
  • Up to 8 internal searches per invocation
