Agent & Tasks

Launch autonomous sub-agents with the Task (Agent) tool.

Parameters

ParameterRequiredDescription
promptYesTask instructions
descriptionYes3-5 word summary
subagent_typeNoAgent specialization
modelNosonnet/opus/haiku
run_in_backgroundNoAsync execution
isolationNo"worktree" for git isolation
cwdNoWorking directory
nameNoAgent name

When to Use

  • Open-ended research
  • Test suite analysis
  • Code review
  • Long-running operations

When Not to Use

  • Reading specific files
  • Simple searches
  • Trivial tasks

Effective Prompts

Complete briefings with: what to do, what has been tried, specific files to examine, desired output format.

TodoWrite

Structured task lists with content, activeForm, and status (pending/in_progress/completed). Use for 3+ step tasks.
