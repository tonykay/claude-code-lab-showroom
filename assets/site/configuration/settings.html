Configuration / Settings

Settings

Configuration through JSON settings files at multiple levels.

Hierarchy

Increasing priority: Plugin defaults → User settings → Project settings → Local settings → Managed settings

Scopes

ScopeFileNotes
Global (User)~/.claude/settings.jsonModel selection, cleanup policies
Project (Shared).claude/settings.jsonPermissions, MCP servers (version controlled)
Local (Personal).claude/settings.local.jsonIndividual overrides (gitignored)
Managed (Enterprise)Platform-specific (MDM, registry, plist)Highest priority, not user-overridable

Open with /config command or edit JSON directly (auto-reloads).

Key Options

  • model — Model selection
  • permissions — Permission rules
  • hooks — Lifecycle hooks
  • cleanupPeriodDays — Session cleanup
  • env — Environment variables
  • language — Language preference
  • alwaysThinkingEnabled — Force thinking

Enterprise

  • allowManagedHooksOnly
  • allowManagedPermissionRulesOnly
