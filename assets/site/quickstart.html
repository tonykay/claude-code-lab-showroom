Getting Started / Quickstart

Quickstart

Get up and running with Claude Code in minutes.

Prerequisites

Requires Node.js 18+ and npm. Check your version:

node --version

If not installed, download from nodejs.org.

Installation

Install globally:

npm install -g @anthropic-ai/claude-code

Verify:

claude --version

Authentication

Run claude to start setup with OAuth browser sign-in. Alternatively:

export ANTHROPIC_API_KEY=sk-ant-...
API keys take priority when the ANTHROPIC_API_KEY environment variable is set.

Project Navigation

cd my-project

Interactive Session

Execute claude to begin work. The system accepts natural language tasks and shows plans before making permission-requiring changes. Example tasks include:

  • Explaining codebases
  • Adding validation
  • Writing tests
  • Finding error handling issues

CLAUDE.md Initialization

Run /init within a session to generate project-specific configuration. The file captures build commands, testing requirements, and team conventions, loading automatically each session.

Non-Interactive Mode

Use the -p flag for single commands:

claude -p "explain this codebase"

Key Slash Commands

CommandDescription
/helpDisplay commands and shortcuts
/initGenerate CLAUDE.md
/memoryView memory files
/permissionsAdjust autonomy settings
/mcpManage MCP servers
/clearReset context
/exitEnd session
