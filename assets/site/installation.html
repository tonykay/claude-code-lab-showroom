Getting Started / Installation

Installation

Claude Code is Anthropic's official CLI tool available for macOS, Linux, and Windows (via WSL), requiring Node.js 18+.

Core Requirements

Node.js 18 or higher with npm included. Verify:

node --version
npm --version

Install

npm install -g @anthropic-ai/claude-code

Verify:

claude --version

Platform-Specific Notes

macOS

Potential permission issues with npm global installs. Use nvm/fnm or configure npm prefix to avoid sudo.

Linux

Avoid sudo for npm installs. Use nvm or adjust npm prefix.

Windows

Must use WSL, not Command Prompt or PowerShell. Steps:

  1. Install WSL
  2. Launch Ubuntu
  3. Install Node.js via nvm
  4. Install Claude Code

Project files should live inside the WSL filesystem.

Maintenance

npm update -g @anthropic-ai/claude-code
# or
claude update

Uninstall:

npm uninstall -g @anthropic-ai/claude-code

Common Issues

  • PATH configuration
  • Outdated Node.js
  • Permission errors
  • Auth failures
  • Docker/CI deployments

Auth can use OAuth or API key from Anthropic Console.
