Concepts / Tools

Tools

Reference for every built-in tool Claude Code can use.

File Tools

Read

Reads files with line numbers (cat -n format). Up to 2000 lines. Supports images, PDFs (20 pages), notebooks. Always auto-approved.

Edit

Exact string replacement. Requires prior Read. Must be a unique match or use replace_all.

Write

Create new files or overwrite existing. Requires prior Read for existing files. Prefer Edit for modifications.

Glob

Find files by pattern. Sorted by modification time. Always auto-approved.

Shell Tool

Bash

Persistent shell session. Compound commands checked independently. Background execution supported. Output truncated if too large.

Search Tools

Grep

ripgrep-based. Three output modes: files_with_matches, content, count. Multiline supported. Always auto-approved.

LS

List directory contents. Always auto-approved.

Web Tools

WebFetch

Fetch URL, convert HTML to markdown, process with secondary model. 15-minute cache. Read-only.

WebSearch

Search web for current info. US-only. Domain filtering. Citations required.

Agent and Task Tools

Task (Agent)

Spawn sub-agents with isolated context. Local or remote execution.

TodoWrite

Structured task lists with statuses.

Notebook Tool

NotebookEdit

Edit Jupyter notebook cells.

MCP Tools

Named mcp__<server>__<tool>. Configured in mcp_servers.json.

Tool availability affected by: CLAUDE_CODE_SIMPLE=1, permission deny rules, isEnabled() checks, MCP server state. Check with /tools command.
