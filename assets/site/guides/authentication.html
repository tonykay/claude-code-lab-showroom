Guides / Authentication

Authentication

How to authenticate Claude Code for different environments.

Default: Claude.ai OAuth

Browser-based login, stores token, auto-refreshes. This is the recommended method.

Alternatives

API Key

Set via ANTHROPIC_API_KEY environment variable or settings file with helper command.

AWS Bedrock

CLAUDE_CODE_USE_BEDROCK=1

Plus standard AWS credentials.

GCP Vertex AI

CLAUDE_CODE_USE_VERTEX=1

Plus Application Default Credentials.

Account Management

  • /login — New OAuth authentication
  • /logout — Remove credentials

Priority Order

ANTHROPIC_AUTH_TOKEN checked first, ANTHROPIC_API_KEY last.
