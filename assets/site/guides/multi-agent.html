Guides / Multi-Agent

Multi-Agent

Sub-agents are independent Claude instances with isolated contexts and customizable system prompts.

Each agent starts fresh, gets a specialized system prompt based on type, and has its own tool permissions.

Use Cases

  • Parallel independent tasks
  • Specialized delegation
  • Background research
  • Solution exploration

Invocation

Agent tool with task description, prompt, agent type, background flag, and isolation settings.

Modes

  • Foreground (default) — Blocking execution
  • Background — Async with notifications

Context

Zero inherited context; prompts must be self-contained.

Worktree Isolation

Safe exploration using git worktree isolation.

Best Practices

  • Independent tasks only
  • Comprehensive prompts
  • Explicit output formats
  • Conservative spawning
