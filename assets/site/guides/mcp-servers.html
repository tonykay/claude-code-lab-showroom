MCP Servers

MCP servers extend Claude Code by connecting to databases, APIs, and file systems.

Adding Servers

  1. CLI: claude mcp add <name> -- <command> [args...]
  2. Config file: --mcp-config flag with JSON
  3. Interactive: /mcp panel

Config Scopes

ScopeFileUse
Project.mcp.jsonTeam-shared
User~/.claude.jsonPersonal, global
Local.claude/settings.local.jsonPer-project personal

Server Types

  • Stdio — Local subprocess
  • HTTP — Remote server
  • SSE — Server-sent events

Management

Enable, disable, reconnect, and view status via the /mcp command.

Safety

Claude prompts before calling any MCP tool.
